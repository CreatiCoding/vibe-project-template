### 데이터 흐름 예시

1. **방문자** → Traefik → Next.js SSG/ISR 페이지 → (캐시된 HTML)
2. **관리자 로그인** → Auth.js → GitHub OAuth → 세션 생성
3. **컨텐츠 작성** → Next.js API → Prisma → PostgreSQL
4. **리소스 업로드** → Next.js API → MinIO (S3 호환) → 공개 URL 반환
5. **컨텐츠 발행** → ISR revalidate → 정적 페이지 재생성
